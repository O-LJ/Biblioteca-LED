
# Biblioteca LED
## By: LJ

**Biblioteca não bloqueante** que adiciona comandos para o controle básico de um LED não RGB.
Adiciona as funções:
-Ligar o LED
-Desligar o LED
-Desligar o LED após algum tempo
-Alternar o Estado do LED
-Piscar o LED continuamente
-Piscar o LED em uma certa frequencia
-Piscar o LED em uma certa frequencia por um certo número de vezes

**O método** `update()` **deve ser chamado continuamente dentro do `loop()`.
